PRA Nighttime Detection Dashboard

Polarization Ratio Analysis for Geomagnetic Anomaly Detection

This dashboard monitors geomagnetic anomalies across INTERMAGNET stations worldwide. Anomalies are detected using multitaper spectral analysis and correlated with earthquake occurrences to assess prediction reliability. Stations are color-coded: gray (normal), orange (anomaly with earthquake correlation), and red (false alarm).

Methodology

Detection Method: Multitaper Spectral Analysis (NW=3.5) + Extreme Value Theory (EVT) + nZ Normalization
Frequency Band: 0.095-0.110 Hz | Time Window: 20:00-04:00 Local Time
Earthquake Correlation: Within 200 km radius, 14 days before anomaly detection
